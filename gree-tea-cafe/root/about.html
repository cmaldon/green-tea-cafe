About

All of the information on this site about tea was provided by Wikipedia.com. The content of this site is fictitious, the Green Tea Cafe does not exist as a business.

These web pages were created by Celeste Maldonado as a project for CS 50B Web Development 2, a class taken at SRJC in Fall 2020.

These pages were made using HTML5, CSS3, JavaScript and Bootstrap.

The images used are from Pikist.com. The font is from GoogleFonts.com. The icons are from FontAwesome.com. The fake map is from RomalCapital.com. I used Pixlr.com to edit the photos.
